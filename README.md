# Power BI Analysis
This project is on creating my first Power BI dashboard. Data is analyzed from the Occupational Employment and Wage Statistics for California report from [Data.gov](https://catalog.data.gov/dataset/occupational-employment-and-wage-statistics-oews-4b4c4). I opted to focus on engineers in order to keep my dashboard simple. I utilized dynamic titles based on slicer selections to accurately describe my analysis. The dashboard can be viewed with this [link](https://app.powerbi.com/view?r=eyJrIjoiODM3OTg3OGUtMjM5MS00YjE1LWE3ODgtZDRkZTYxY2YyNTMyIiwidCI6IjI1YjY4NDY5LWY0MWItNGU0OC1iYmMwLTQ5OTU1Yzc4YmY5MCIsImMiOjZ9).
